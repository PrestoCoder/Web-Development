Block elements vs Inline elements:-
Block:- Takes the entire viewport width(eg:- p, div, li, ul, ol)
Inline:- Takes the space that is required(eg:- span)

List: 
    -unordered list(ul)

Attributes:- For an element, attributes are qualities that we can alter
Like for ol, type and start are attributes. start takes numerical value always.
